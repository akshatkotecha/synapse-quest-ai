# Synapse Quest – AI Powered Developer Productivity Platform

Synapse Quest is a developer productivity analytics platform designed to analyze developer behavior and optimize task assignment within engineering teams.

The system collects developer activity signals and evaluates productivity metrics to provide insights into performance patterns and potential workflow inefficiencies.

This project was initially developed during a hackathon and later extended with additional analytics and monitoring capabilities.

---

## Features

- Developer productivity tracking using Git activity telemetry
- AI-driven task routing based on developer expertise
- Behavioral analytics for engineering teams
- Gamified leaderboard and XP reward system
- REST API endpoints for developer analytics

---

## System Overview

Synapse Quest analyzes developer activity across repositories and tasks to generate behavioral insights such as:

- Activity levels
- Task completion speed
- Developer engagement patterns
- Productivity trends

These insights can help engineering teams identify potential bottlenecks and optimize task distribution.
